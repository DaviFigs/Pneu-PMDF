Sobre o software


Este software foi desenvolvido para armazenar e registrar de forma fácil e organizada, registros dos pneus de viaturas da PMDF retirados dos armazéns,os pedidos são feitos a diversos batalhões da Polícia militar espalhados pelo Distrito Federal.
As funcionalidades aparecem na barra superior do software, de acordo com as permissões do usuário que esta logado.

SITE OFICIAL DA PMDF

Registros


Os registros de pedidos feitos aos batalhões são efetuados por apenas um tipo de usuário com cargo de registrador, os demais usuários podem apenas visualizar os pedidos já efetuados.
Para registrar, o usuário registrador deve selecionar todos os dados correspondentes à aquele pedido, como o batalhão.
Pedidos são organizados por data.

  • Código do pedido/PIM, (Proveniente de outro departamento)
  • Viatura a qual foram fornecidos pneus
  • Quantidade de pneus
  • Data do pedido (automatica)

No registro, apenas com a informação de para qual viatura o pedido está sendo feito, temos todas as outras informações necessárias. Ex:Tipo de pneu que ela usa, nome do carro, prefixo, tipo de carro, etc.

Visualização


Todos os pedidos antes registrados estão disponíveis para serem visualizados em forma de tabelas, todos os usuários logados no sistema podem visualiza-los apenas clicando nos botões disponíveis na barra de navegação acima.
